# PIXI Letters
Typing keyboard game with falling letters.

DEMO [Mental](https://mental.gatsbyjs.io/pixi-letters.html)

## Quick start

1. **Start developing.**

    ```shell
    nvm use 16.17.1
    yarn
    yarn start
    ```

2. **Open the source code and start editing.**

   Your site is now running at `http://localhost:3032`!

## Features

1.  PIXI.js
2.  Webpack
3. OOP
